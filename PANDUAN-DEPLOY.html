Install Tools di Komputer
Hanya perlu dilakukan sekali

Buka Terminal (Mac/Linux) atau Command Prompt (Windows) lalu jalankan perintah ini satu per satu:

1️⃣ Install Node.js (download dari nodejs.org, pilih LTS)
# Cek apakah Node.js sudah terinstall:
node --version
# Harus muncul: v18.x.x atau v20.x.x
2️⃣ Install Git (download dari git-scm.com)
git --version
# Harus muncul: git version 2.x.x
3️⃣ Install Vercel CLI
npm install -g vercel
vercel login
# Akan terbuka browser → login dengan GitHub
2
Setup Project Weiiz.ink
Download dan install semua dependensi
Download starter kit dan install
# Masuk ke folder kerja kamu (contoh: Documents)
cd Documents

# Extract folder weiiz-ink yang sudah didownload
# Lalu masuk ke folder tersebut:
cd weiiz-ink

# Install semua package (tunggu 1-3 menit)
npm install
Buat file environment variabel
# Copy file contoh menjadi .env.local
# Mac/Linux:
cp .env.example .env.local

# Windows:
copy .env.example .env.local
⚠️
Penting!
File .env.local berisi password dan secret key. Jangan pernah upload file ini ke GitHub. File .gitignore sudah dikonfigurasi untuk mengabaikannya secara otomatis.
3
Setup Database (Neon.tech)
Database PostgreSQL gratis untuk menyimpan semua data
  1. Buka neon.tech → Sign up → Create Project
  2. Beri nama project: weiiz-ink
  3. Setelah project dibuat, klik Dashboard
  4. Klik "Connection string"
  5. Pilih format: Prisma
  6. Copy string yang muncul (dimulai dengan postgresql://...)
Buka .env.local dan isi DATABASE_URL
# Di file .env.local, ubah baris ini:
DATABASE_URL="postgresql://user:password@host/weiiz?sslmode=require"

# Ganti dengan string yang kamu copy dari Neon:
DATABASE_URL="postgresql://raka:Abc123@ep-xxx.us-east-2.aws.neon.tech/weiiz?sslmode=require"
Push schema database (buat semua tabel)
# Generate Prisma client dulu
npm run db:generate

# Push schema ke database (buat semua tabel)
npm run db:push

# Kalau berhasil akan muncul:
# ✅ Your database is now in sync with your Prisma schema.
Opsional: Lihat database secara visual
npm run db:studio
# Buka browser di http://localhost:5555
# Kamu bisa lihat dan edit data langsung

Setup Midtrans (Payment Indonesia)
Terima pembayaran: QRIS, GoPay, OVO, Transfer Bank
  1. Buka dashboard.midtrans.com → Daftar akun
  2. Setelah login, pastikan pilih mode Sandbox (untuk testing)
  3. Klik Settings → Access Keys
  4. Copy Server Key dan Client Key
Isi di .env.local
MIDTRANS_SERVER_KEY="SB-Mid-server-xxxxxxxxxxxxxxxxxxxxxxxx"
MIDTRANS_CLIENT_KEY="SB-Mid-client-xxxxxxxxxxxxxxxxxxxxxxxx"
MIDTRANS_IS_PRODUCTION="false"
# Ganti ke "true" nanti setelah akun Midtrans diverifikasi
🧪
Kartu Test untuk Sandbox
Untuk testing pembayaran, gunakan kartu virtual ini:
Nomor: 4811 1111 1111 1114  ·  Expired: 01/25  ·  CVV: 123
Atau pilih "GoPay Simulator" saat checkout.

Setup Email (Resend)
Kirim email otomatis: verifikasi, konfirmasi pembelian, notifikasi
  1. Buka resend.com → Sign Up (bisa pakai GitHub)
  2. Klik API Keys → Create API Key
  3. Beri nama: weiiz-production
  4. Copy API Key yang muncul (dimulai re_...)
Isi di .env.local
RESEND_API_KEY="re_xxxxxxxxxxxxxxxxxxxxxxxx"
EMAIL_FROM="Weiiz.ink <noreply@weiiz.ink>"
# Untuk testing, kamu bisa pakai email kamu sendiri dulu
📧
Testing Email Lokal
Di mode development, Resend akan mengirim email ke alamat yang kamu daftar. Untuk production, kamu perlu verifikasi domain di dashboard Resend. Gratis untuk 3.000 email/bulan.

8
Deploy ke Vercel
Publish app kamu ke internet — gratis!
  1. Buka vercel.com → Login dengan GitHub
  2. Klik Add New → Project
  3. Pilih repository weiiz-ink
  4. Vercel otomatis detect Next.js — klik Deploy
  5. TUNGGU — jangan tutup dulu! Build akan gagal karena ENV belum diisi
  6. Setelah failed, klik Settings → Environment Variables
🔑
Tambahkan Semua Environment Variables
Di Vercel Settings → Environment Variables, tambahkan satu per satu semua variable dari file .env.local kamu. Ini wajib dilakukan agar app bisa berjalan.
Environment variables yang harus diisi di Vercel:
DATABASE_URL
Connection string PostgreSQL dari Neon.tech
JWT_SECRET
String acak 32 karakter untuk enkripsi login. Generate di: generate-secret.vercel.app/32
NEXT_PUBLIC_APP_URL
URL production kamu, misal: https://weiiz-ink.vercel.app
MIDTRANS_SERVER_KEY
Server Key dari dashboard Midtrans
MIDTRANS_CLIENT_KEY
Client Key dari dashboard Midtrans
MIDTRANS_IS_PRODUCTION
Isi false untuk sandbox, true untuk production
RESEND_API_KEY
API Key dari resend.com
EMAIL_FROM
Weiiz.ink <noreply@weiiz.ink>
Setelah semua ENV diisi, redeploy:
# Di terminal:
vercel --prod

# Atau di dashboard Vercel: klik "Redeploy"
9
Setup Webhook Midtrans
Agar status pembayaran terupdate otomatis

Webhook adalah URL yang dipanggil Midtrans setiap kali ada pembayaran. Tanpa ini, status transaksi tidak akan otomatis berubah.

  1. Buka dashboard.midtrans.com
  2. Klik Settings → Configuration
  3. Isi Payment Notification URL dengan:
# Ganti dengan URL Vercel kamu:
https://weiiz-ink.vercel.app/api/payment/webhook

# Atau kalau pakai domain sendiri:
https://weiiz.ink/api/payment/webhook
  1. Isi juga Finish Redirect URL: https://weiiz-ink.vercel.app/payment/success
  2. Klik Save
🎯
Test Webhook
Di Midtrans sandbox, buat transaksi test. Setelah bayar, cek di database Neon → tabel Transaction apakah status berubah menjadi SUCCESS. Kalau ya, webhook berhasil!
10
Setup Domain Custom (Opsional)
Pakai weiiz.ink atau domain kamu sendiri

Tanpa domain custom, app kamu tetap bisa diakses di nama-project.vercel.app — sudah cukup untuk mulai.

  1. Beli domain di Niagahoster/Dewaweb/Namecheap (contoh: weiiz.ink)
  2. Di Vercel: Project Settings → Domains → Add Domain
  3. Ketik domain kamu lalu klik Add
  4. Vercel akan tampilkan DNS record yang harus ditambahkan
  5. Di panel DNS domain kamu, tambahkan record tersebut:
# DNS Record yang perlu ditambahkan:
Type: A
Name: @ (atau kosong)
Value: 76.76.21.21

Type: CNAME
Name: www
Value: cname.vercel-dns.com
⏱
DNS Propagation
Setelah mengubah DNS, tunggu 15 menit sampai 24 jam untuk domain aktif. Ini normal dan tidak bisa dipercepat. Vercel akan otomatis beri SSL certificate gratis.

🔧 Troubleshooting Umum
❌ Error: "Can't reach database server"
→ Cek DATABASE_URL di .env.local sudah benar
→ Pastikan ada ?sslmode=require di akhir URL
→ Jalankan npm run db:push ulang
❌ Error: "JWT_SECRET is not defined"
→ Pastikan .env.local ada dan berisi JWT_SECRET
→ Di Vercel, pastikan ENV sudah ditambahkan dan sudah redeploy
❌ Pembayaran tidak update statusnya
→ Cek webhook URL di Midtrans sudah benar
→ Pastikan URL production (bukan localhost) yang diisi
→ Cek Vercel Function Logs untuk error di webhook handler
❌ Build gagal di Vercel
→ Klik "View Build Logs" di Vercel untuk lihat error spesifik
→ Paling sering karena ENV variable belum diisi
→ Pastikan semua ENV dari .env.local sudah ditambahkan di Vercel
